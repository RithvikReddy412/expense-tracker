# expense-tracker
an expense tracker with local storage.

A web-based Expense Tracker built with HTML, CSS, and JavaScript that helps users manage income and expenses efficiently.

## Features

- Add income and expense entries
- View transaction history
- Track total balance
- Display income and expense summaries
- Delete transactions
- Responsive design

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Getting Started

1. Clone the repository.
2. Open `index.html` in your browser.
